# 🚀 Projeto de Carreira em Tecnologia

Este repositório documenta uma experiência prática utilizando **inteligência artificial como agente de carreira e entrevistador de desempenho**.  
O objetivo foi simular uma entrevista estruturada para identificar a área de tecnologia mais adequada e, a partir disso, construir um **plano personalizado de estudos e desenvolvimento profissional**.

Durante o processo, foram analisados:
- Interesses pessoais (dados e inteligência artificial)  
- Disponibilidade de tempo para estudo (18h semanais)  
- Objetivo profissional (primeiro emprego em tecnologia)  
- Preferência de atuação (trabalhar com dados)  

O resultado foi a escolha da carreira de **Cientista de Dados**, com um roadmap de 90 dias, projeto de portfólio e roteiro de entrevistas para apoiar a entrada no mercado.

# 📊 Resumo da Entrevista de Carreira em Tecnologia

Este projeto foi realizado com apoio de uma **inteligência artificial atuando como agente de carreira e entrevistador de desempenho**.  
O objetivo foi identificar a melhor área de tecnologia para iniciar a jornada profissional e estruturar um plano de desenvolvimento.

## 📝 Etapas da Entrevista
1. Identificação dos interesses principais (resolver problemas, criar produtos ou entender sistemas).  
2. Verificação da experiência prévia em tecnologia.  
3. Disponibilidade semanal para estudos.  
4. Preferência de atuação (pessoas, dados ou código).  
5. Objetivo profissional (primeiro emprego, transição ou crescimento).  
6. Áreas de maior interesse (ex: dados, inteligência artificial).  
7. Experiência prévia fora da tecnologia que poderia ser aproveitada.  

## 🎯 Resultado da Análise
Com base nas respostas, o agente identificou três carreiras promissoras:  
- 🥇 Cientista de Dados  
- 🥈 Analista de Dados  
- 🥉 Engenheiro de Machine Learning  

A carreira escolhida foi **Cientista de Dados**, considerando afinidade com dados e inteligência artificial, objetivo de primeiro emprego e disponibilidade de 18h semanais para estudo.

## 📦 Plano de Desenvolvimento
O agente de carreira gerou um plano completo com:  
- **Visão do dia a dia** da profissão  
- **Mapa de skills essenciais e complementares**  
- **Roadmap de 90 dias** com metas semanais  
- **Projeto de portfólio** (Previsão de preços de imóveis com Machine Learning)  
- **Roteiro de entrevistas** com perguntas comuns e respostas estruturadas  
- **Trilha recomendada na DIO** (Bootcamp Cientista de Dados com Python)
